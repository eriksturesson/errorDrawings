## ECMAScript Modules

Use the error object "e" as parameter in "errorDrawing(e)"

```
import errorDrawing from 'error-drawings';

async function myExample(){
try{
    myfunctionThatWillReturnError()
} catch(e){
    errorDrawing(e)
    }
}

```
